In 2019-11-13, Zhun Guo generated some IOP files from Po-lun Ma's gustiness compset FC5AV1C-04P2. 
How to make global IOP files for replaying? 

1 get the master brach of E3SM. git clone https://github.com/E3SM-Project/E3SM.git (The current IOP files are generated by this commit https://github.com/E3SM-Project/E3SM/commit/f10baa68e3578ba466c7cffe0385496a679e10d0)
2 modify run_e3sm_anvil_gust.csh, and csh run_e3sm_anvil_gust.csh

3 To make a single conlum forcing file. 
  3.1 Run diagnostic_v2_0.
  3.2 In folder of diagnostic_v2_0, you will found a folder named 'data' in which there is a nc-file (i.e. casename.ne30_ne30_site_location.nc). Then nudump it.
      For example you will get.
